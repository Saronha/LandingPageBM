# BodyMatch - Landing Page

## Objetivos

Landing page desenvolvida para BodyMatch, startup de moda em fase de pré-incubação pelo Porto Digital (Recife - PE), com foco em conversão de leads.

## Deve conter:

- Apresentação geral do produto oferecido pela bodymatch;
- Design responsivo;
- Sessão de call to action, onde os interessados possam deixar seu e-mail.

## Tecnologias utilizadas

A landing page está a ser desenvolvida apenas com HTML5, CSS3 e JavaScript, optando por uma stack simples e sem dependências externas.
